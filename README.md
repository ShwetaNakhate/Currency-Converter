# Currency Converter 💱

A real-time currency converter built with HTML, CSS, and JavaScript using the ExchangeRate API.

## Features
- Live exchange rates via ExchangeRate API
- Swap currencies with one click
- Supports 12 currencies: USD, INR, EUR, GBP, JPY, AUD, CAD, CHF, CNY, AED, SGD, MXN
- Loading state while fetching rates
- Error handling for network failures

## Tech Used
HTML | CSS | JavaScript

## How to Run
Just open `index.html` in your browser. No setup needed.

## API Used
[ExchangeRate-API](https://www.exchangerate-api.com/)
